# Traveling salesman problem

## Usage

**Input**
```
    n s (n is number of cities, s is start city)
    Matrix n x n. 
    
    Example
    5 0
    0 1 2 7 5 
    1 0 4 4 3
    2 4 0 1 2
    7 4 1 0 3
    5 3 2 3 0
```

**Output**

```
    -1, if there are no ways to go
    cost, the minimun cost 
```
